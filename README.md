# 建筑工程维护系统 (Construction Maintenance System)

这是一个基于 Python Flask + SQLite + Jinja2 + openpyxl 独立构建的轻量级、高品质建筑工程公司综合信息维护系统。本系统完全满足商业级美学标准，采用了精美的“深邃翠绿主题（Premium Emerald Theme）”，提供了平滑的过渡动画与极佳的用户交互体验。

## 🌟 系统核心功能

- **📊 数据看板 (Dashboard)**：
  - 动态计算并展示**本月项目支出**、**累计项目支出**、**月凭证数量**、**批量待确认资料**和**企业资质到期**等关键指标卡片。
  - 提供**项目支出排行榜**与**费用科目构成排行**两个高级分析统计图表。
- **📁 项目台账 (Projects)**：
  - 支持新增并录入工程项目（包含项目名、负责人、开工日期、状态、备注）。
  - 支持以明晰的 HSL 颜色 Badge 标识项目状态（“进行中”、“已完工”、“已暂停”）。
- **✍️ 凭证录入 (Vouchers)**：
  - 支持记录并保存每笔项目支出（包括日期、关联项目、科目类型、金额、录入人、备注），自动格式化金额并汇总。
- **👥 人员花名册 (Personnel)**：
  - 记录进场施工人员的完整档案（姓名、身份证号、性别、年龄、工种、电话、银行卡号及开户行）。
  - 支持导出规范格式的人员信息 Excel。
- **🛡️ 企业资质 (Qualifications)**：
  - 登记各合作公司的资质证书，记录证书编号、发证时间、证书到期时间或是否长期有效。
- **📥 批量录入 (Batch Import)**：
  - 支持通过美观的虚线拖拽区域**批量选择并上传**凭证发票、身份证件等扫描件文件。
  - 引入了待确认任务队列，为未来的 AI / OCR 解析打下基础。
- **📤 导出中心 (Export Center)**：
  - 提供一键导出**项目台账 Excel**、**人员花名册 Excel**和**企业资质证书 Excel**。

## 🛠️ 本地运行与部署

### 1. 创建虚拟环境并安装依赖

在 Windows 系统的 PowerShell 下：

```powershell
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
.\.venv\Scripts\Activate.ps1

# 安装开发包及其依赖
pip install -e ".[dev]"
```

### 2. 启动开发服务器

```powershell
flask --app construction_maintenance run --debug
```

启动后在浏览器中打开 `http://127.0.0.1:5000` 即可享受高颜值的深绿主题 CAM 系统！
